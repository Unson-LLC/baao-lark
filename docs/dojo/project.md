## 01_strategy｜戦略骨子

### プロダクト概要（1文）
毎週のAIアップデートを**実務の言葉**で解釈し、次の一手まで落とし込む会員制コミュニティ。

### ICP（1行）
AI活用に関心が高い事業責任者・リードエンジニア層で、自社へのAI導入を検討している中小〜ミッドマーケット企業の意思決定者。

### 約束する価値（1文）
最新の潮流を「社内で転送できる形」で掴み、PRIME・CONSULTで合意形成、BOOST・RECIPESで実装まで一気通貫で支援。

### プログラム体系

#### 継続課金プログラム

| プログラム | 目的 | 主な提供 | 月額（税抜） | 年額（税抜） |
|-----------|------|---------|------------|------------|
| **稽古場 BASIC** | 最新と実務解釈を毎週アップデート | 30分解説＋30分Q&A、ノート、アーカイブ | **2,980円** | **29,800円** |
| **稽古場 ACTIVE** | BASIC＋過去アーカイブ全文検索 | BASIC特典＋全アーカイブ＋Flow別フィード | **9,800円** | **98,000円** |
| **師範会 PRIME** | 先行事例・見学で意思決定を加速 | クローズド会、現場見学、ピアレビュー、**全サービス20%割引** | **196,000円** | **200万円** |

#### 都度課金サービス（PRIME会員は20%割引）

| サービス | 目的 | 料金（税抜） | PRIME価格 |
|---------|------|------------|----------|
| **CONSULT クイック** | 単発相談（60分×1回） | **15万円** | **無料** |
| **CONSULT スタンダード** | 稟議支援（90分×2回＋稟議素材） | **35万円** | **28万円** |
| **型録 RECIPES** | そのまま配れる業務手順書 | **100–150万円/本** | **80–120万円** |
| **立合い BOOST** | 4–8週で可視ROI | **400–800万円** | **320–640万円** |

> **CONSULT→BOOST移行特典**: CONSULT実施後3ヶ月以内にBOOSTを契約した場合、CONSULT料金をBOOSTから全額差し引き

### ファネル設計

```
Visitor（無料）
    ↓ 週次ライブ体験
BASIC参加（月2,980円）
    ↓ もっと深掘りしたい
ACTIVE参加（月9,800円）
    ↓ 事例に興味・合意形成が必要
PRIME入会（年200万円）+ 20%割引獲得
    ↓ 具体的に相談したい
CONSULT（クイック/スタンダード）
    ↓ 本格導入したい
RECIPES活用（手順書） または BOOST実施（プロジェクト）
    ↓ 成果が出た
PRIMEで事例共有・次の勝ち筋へ
    ↓
ケース公開・横展開
```

## 02_offer｜価格・オファー

### BASIC｜稽古場 AI Weekly

**月額2,980円 / 年額29,800円**

| 提供内容 | 詳細 |
|---------|------|
| 週次ライブ | 30分解説＋30分Q&A（アーカイブ有） |
| BASICノート | 1枚サマリPDF |
| MYWA Picks | Top10リンク集 |
| Field別フィード | 流派ごとの今週の示唆と次アクション |
| PRIME特典 | アーカイブ1本無料視聴 |

**運営ルール**:
- ライブは会員限定、翌週に一部一般公開（鮮度管理）
- 参加者の質問は匿名化してログ化
- 1枚サマリの社内転送可、録画の再配布不可

### ACTIVE｜稽古場 Weekly + Archive

**月額9,800円 / 年額98,000円**

BASIC特典に加えて:
- 過去アーカイブ全文検索
- Flow別フィード（流派ごとのディープコンテンツ）
- 月次イベント参加

### PRIME｜師範会 Executive Community

**月額196,000円 / 年額200万円**

- クローズド会（経営層限定）
- 現場見学（先進企業訪問）
- ピアレビュー
- **全サービス20%割引**
- CONSULT クイック無料

## 03_sales_ops｜営業オペレーション

### ターゲット優先度
1. **研修参加者**（キャリアパートナーズ経由など）→ BASIC
2. **メディア経由**（ラジオ、書籍、SNS）→ BASIC
3. **紹介・口コミ** → ACTIVE/PRIME
4. **CONSULT経由** → PRIME/BOOST

### 転換施策

| 転換 | 施策 |
|------|------|
| FREE→BASIC | 週次ライブ体験、初月無料キャンペーン |
| BASIC→ACTIVE | Flow別フィード訴求、アーカイブ検索体験 |
| ACTIVE→PRIME | 見学招待（1回無料）、Executive Desk無料枠 |
| PRIME→BOOST | BOOSTプレビュー（90分無料）、CONSULT料金差し引き |

### チャネル連携

| チャネル | Dojo誘導ポイント |
|---------|-----------------|
| 研修事業 | 研修終了時にBASIC案内 |
| ラジオ | 番組内でDojo告知 |
| 書籍 | 巻末・QRコード |
| SNS | 定期投稿 |
