## 00_reality_check｜事業の現実認識

### この事業の本質的な問い

1. **誰のPLなのか？**
   - BAAO法人として収益を上げるのか
   - 山本個人の講師業として稼ぐのか
   - Unsonへのリード獲得チャネルとして機能させるのか
   → **現状未決定。初回セミナー後に決める。**

2. **キャリアパートナーズ依存の構造**
   - リード獲得: キャリアパートナーズ
   - 集客・告知: キャリアパートナーズ
   - 運営事務: キャリアパートナーズ
   - 請求・入金: キャリアパートナーズ
   → **BAAOは「講師派遣 + コンテンツ供給」に近い立場**

3. **「研修屋」から抜けられるか？**
   - 現状の見え方: 「AIガバナンスに強い、よくある生成AI研修」
   - 差別化の根拠: 山本の権威性（経産省ガイドライン執筆協力）
   - 課題: 権威性が「山本個人」に紐づいており、BAAOのプロダクトになっていない

### 致命的に足りていないもの

1. **ユニットエコノミクス**: 1案件でBAAOにいくら残るか不明
2. **顧客所有権**: キャリアパートナーズ経由の顧客はBAAO/Unsonでフォローできるのか
3. **講師育成計画**: 山本依存からの脱却プランなし
4. **LTV設計の数値根拠**: ファネル各段階のコンバージョン率が楽観的

## 02_business_model｜ビジネスモデルの選択肢

### Option A: 講師派遣モデル（現状の延長）

```
キャリアパートナーズが顧客開拓・運営
    ↓
BAAOは講師（山本）を派遣
    ↓
売上の40〜60%がBAAO取り分
```

**メリット**:
- 営業・運営負荷がほぼゼロ
- キャリアパートナーズの顧客基盤を活用

**デメリット**:
- 顧客リストはキャリアパートナーズ所有
- Dojo/Unson送客の自由度が低い
- 価格決定権がない
- スケールしにくい（山本の稼働上限）

**向いているケース**:
- 山本の講師業としての副収入
- 事業としてではなく個人の活動

### Option B: 自社事業モデル（独立路線）

```
BAAO自身がリード獲得・運営
    ↓
売上100%がBAAO
    ↓
顧客リストをBAAO/Unsonで活用
```

**メリット**:
- 顧客所有権がBAAO
- Dojo/Unson送客を自由に設計
- 価格決定権がBAAO

**デメリット**:
- 営業・運営の負荷が大きい
- リード獲得チャネルをゼロから構築
- 初期投資が必要

**向いているケース**:
- Unsonへのリード獲得チャネルとして本気でやる場合

### Option C: ハイブリッドモデル（推奨）

```
Phase 1: キャリアパートナーズ経由で実績構築
    ↓
Phase 2: 実績を武器にBAAO独自チャネルを開拓
    ↓
Phase 3: キャリアパートナーズ依存度を下げつつ両立
```

**条件**:
- キャリアパートナーズ経由の顧客にもDojo案内が可能（合意必須）
- 研修参加者リストの共有範囲を明確化

## 03_partner_negotiation｜キャリアパートナーズとの交渉ポイント

### 合意すべき事項

| 項目 | BAAO希望 | 交渉優先度 |
|------|---------|-----------|
| **収益分配率** | BAAO 60%以上 | 高 |
| **顧客リスト共有** | 研修参加者リストをBAAOにも提供 | 最高 |
| **Dojo案内の許可** | セミナー終了時にDojo案内可 | 最高 |
| **追加営業の許可** | 研修後のBAAO/Unson営業可 | 高 |
| **価格決定権** | 有料研修の価格はBAAOが決定 | 中 |
| **独自開催の許可** | BAAO単独でのセミナー開催可 | 中 |

### 交渉の優先順位

1. **顧客リストとフォロー権限**（これがないとUnson送客の意味がない）
2. **収益分配率**（60%以上確保）
3. **Dojo案内の許可**

### 交渉が決裂した場合

- キャリアパートナーズ経由は「山本の講師業」として割り切る
- BAAO事業としては独自チャネル構築に集中

## 04_pricing｜価格設計（シンプル版）

### 有料研修メニュー（2パターンに絞る）

| メニュー | 価格 | 対象 | 内容 |
|---------|------|------|------|
| **基本研修** | 50万円/回（20名まで） | 一般社員向け | AI概論+基本操作（3h） |
| **管理職研修** | 80万円/回（15名まで） | 管理職向け | ガバナンス+組織導入（3h） |

**追加オプション**:
- 追加人数: +2万円/人
- ハンズオン追加: +30万円（半日延長）
- 出張対応: 交通費・宿泊費実費

### 助成金について（現実的な対応）

**やること**:
- 「人材開発支援助成金の対象になる可能性があります」と情報提供
- 助成金申請の流れを簡単に説明した資料を用意

**やらないこと**:
- 助成金申請代行
- 「申請コンサル込み」の訴求

**理由**: 助成金申請は別事業レベルの負荷。提携社労士への紹介に留める。

## 07_trainer_development｜講師育成計画

### 現状の課題

- 山本以外に講師がいない
- 「ガバナンス権威性」が山本個人に紐づいている
- 山本が倒れたら事業終了

### 育成ロードマップ

| フェーズ | 時期 | 目標 | 具体策 |
|---------|------|------|--------|
| Phase 1 | 2026年Q1 | 候補者選定 | 佐藤/卯田以外で1名特定 |
| Phase 2 | 2026年Q2 | 部分担当 | 基礎操作パート（1h）を任せる |
| Phase 3 | 2026年Q3 | 単独登壇 | 基本研修を単独で担当可能に |
| Phase 4 | 2026年Q4 | 認定制度 | 「BAAO認定AI研修講師」制度化 |

### 講師候補の条件

- AI活用の実務経験（3年以上）
- プレゼンテーション能力
- BAAOの価値観に共感
- 山本の監修を受けられる関係性

### 権威性の分散

| 権威性の源泉 | 現状 | 目標状態 |
|-------------|------|---------|
| 経産省ガイドライン | 山本個人 | 「BAAO監修」として法人に帰属 |
| 研修実績 | 山本100回 | 「BAAOとして累計◯回」 |
| メディア露出 | 山本個人 | BAAO法人としての露出を増やす |
